# tv-detection-project
Проект для автоматического подсчета телевизоров с использованием YOLOv8
TV Detection Project

Проект для автоматического подсчета телевизоров с использованием YOLOv8.

Запуск:
ВАЖНО: перед запуском всего проекта требуется установить на компьютер Python 3.8 или выше.
1) Открыть командную строку (терминал)
2) В командной строке перейти в директорию распоковки проекта ... tv-detection-project
3) Ввести в командной строке поочерёдно следующие команды и дождаться их полного выполнения:
	pip install -r requirements.txt
	pip install openpyxl
"Эти команды установят необходимые зависимости для запуска приложения"
4) Ввести в командной строке следующую команду:
	python app.py
"Эта команда запустит приложение, а также запустит сервер на локальном адресе 127.0.0.1:5000"
5) Открыть браузер и ввести в адресную следующее:
	http://127.0.0.1:5000
"Откроется страничка приложения с простым веб-интерфейсом по подсчёту количества телевизоров"
6) В веб-интерфейсе нажмите на кнопку "Выберите файл" и выберите файл-картинку для подсчёта телевизоров 
	ВАЖНО: название файла должно состоять из латинских букв и/или арабских цифр
7) После загрузки файла нажмите кнопку "Обработать" для вывода результата
8) На экране отобразится:
	надпись "Найдено телевизоров: " ; 
	высветится их количество; 
	жёлтой рамкой обведутся подсчитанные программой телевизоры; 
	в файле history.json добавится запись с датой, названием файла и результатом;
	Появится кнопка "Скачать Excel отчет", в котором также отобразятся дата, название файла и результат
